# MERN AI Flow App

## Tech Stack

MongoDB
Express
React
Node
React Flow
OpenRouter API

## Setup Instructions

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev

## Environment variables

Create backend/.env:
